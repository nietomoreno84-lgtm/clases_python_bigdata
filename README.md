## 🚀 Cómo empezar

### 1. Clona el repositorio (solo la primera vez)

```bash
git clone <URL_DEL_REPOSITORIO>
```

### 2. Actualiza tu copia local (cada vez que el profesor suba contenido nuevo)

```bash
git pull
```

---

## ⚠️ Importante

Este es un repositorio **de solo lectura para los alumnos**. Esto significa que:

- ✅ Puedes clonarlo y descargarlo en tu equipo
- ✅ Puedes hacer `git pull` para recibir las actualizaciones del profesor
- ❌ No tienes permisos para hacer `push` ni modificar el repositorio remoto

> Si quieres guardar tus propias versiones o practicar con el código, te
> recomendamos crear tu propio repositorio en tu cuenta personal.

---

## 🔔 ¿Cómo saber cuándo hay contenido nuevo?

El profesor avisará en clase o por los canales habituales de comunicación cuando
suba material nuevo. Cuando eso ocurra, simplemente ejecuta `git pull` dentro de
la carpeta del repositorio para sincronizar los cambios.
